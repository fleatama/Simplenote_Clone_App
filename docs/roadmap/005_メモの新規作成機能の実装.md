# 5. メモの新規作成機能の実装

ユーザーが新しいメモを作成できる機能を実装した。

## 主な作業内容

1.  **イベントハンドラの作成:**
    - `app/page.tsx` に `handleCreateNewNote` という非同期関数を定義。
    - この関数は、`/api/notes` エンドポイントに対して `POST` リクエストを送信する。

2.  **UIへの接続:**
    - 左ペインの「新規作成」ボタンに `onClick` イベントを追加し、`handleCreateNewNote` 関数を呼び出すようにした。
    - デバッグ: `onClick` の設定漏れによりボタンが反応しない問題を `console.log` を使って特定し、解決した。

3.  **状態の更新:**
    - `POST` リクエストが成功し、新しいメモのデータがAPIから返ってきたら、Reactの状態更新関数 `setNotes` を使って、既存のメモリストの**先頭**に新しいメモを追加。
    - これにより、画面上のメモ一覧がリアルタイムで更新され、新しく作成したメモが一番上に表示されるようになった。
