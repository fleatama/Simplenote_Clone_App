# 4. メモ一覧の表示機能の実装

バックエンドAPIから取得したメモのリストを、画面左側のペインに表示する機能を実装した。

## 主な作業内容

1.  **クライアントコンポーネント化:**
    - `app/page.tsx` の先頭に `"use client";` を宣言。ブラウザ上でインタラクティブに動作するコンポーネントに変更した。

2.  **状態管理とデータ取得:**
    - Reactの `useState` フックを使い、メモのリスト (`notes`)、ローディング状態 (`loading`)、エラー状態 (`error`) を管理する変数を定義した。
    - Reactの `useEffect` フックを使い、ページが最初に表示された時に一度だけ `/api/notes` エンドポイントに `fetch` リクエストを送信し、取得したメモデータを `notes` 状態に保存するようにした。

3.  **リストの描画:**
    - 取得した `notes` データを `map` 関数でループ処理し、Bootstrapの `list-group` を使って各メモのタイトルと更新日時を一覧表示するようにした。
    - ローディング中やエラー発生時の表示も実装した。
