# 9. Vercel/Netlifyへのデプロイ

開発したアプリケーションをインターネット上に公開し、誰でもアクセスできるようにする。

## 実装予定の作業

1.  **Gitリポジトリの準備:**
    - これまで開発してきたコードを、最終的に `main` ブランチにマージし、GitHubにプッシュする。

2.  **Vercel/Netlifyへのサインアップと連携:**
    - 無料プランが利用できるVercelまたはNetlifyにサインアップする。
    - GitHubアカウントと連携し、デプロイしたいリポジトリ (`Simplenote_Clone_App`) を選択する。

3.  **ビルド設定とデプロイ:**
    - Next.jsプロジェクトであることをプラットフォームが自動で認識するため、基本的なビルド設定は不要な場合が多い。
    - 「Deploy」ボタンを押し、ビルドとデプロイが自動的に実行されるのを待つ。

4.  **動作確認:**
    - デプロイが完了すると発行される公開URLにアクセスし、アプリケーションがオンラインで正しく動作することを確認する。
    - 今後の開発で `main` ブランチにプッシュするたびに、自動で最新版がデプロイされるようになる。
