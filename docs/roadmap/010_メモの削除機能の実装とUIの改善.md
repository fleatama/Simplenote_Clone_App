# 10. メモの削除機能の実装とUIの改善

不要なメモを削除できる機能を追加し、あわせてボタン類をテキストからアイコンへとアップグレードした。

## 主な作業内容

1.  **DELETE APIの活用:**
    - すでに作成してあった `/api/notes/[id]` の `DELETE` エンドポイントをフロントエンドから呼び出す `handleDeleteNote` 関数を実装。
    - 削除前に「本当に削除しますか？」と確認するアラートを表示するようにした。

2.  **Bootstrap Iconsの導入:**
    - `npm install bootstrap-icons` でライブラリを導入。
    - `app/layout.tsx` にCSSをインポートし、プロジェクト全体でアイコンを使用可能にした。

3.  **UIのアイコン化:**
    - 「新規作成」「削除」「タイムスタンプ挿入」の3つのボタンを、それぞれ `bi-plus-lg`, `bi-trash`, `bi-clock` アイコンに置き換え。
    - Flexbox (`d-flex`) を使い、アイコンがボタンの中央に配置されるよう調整した。
    - アイコンだけでは意味が伝わりにくい場合に備え、`title` 属性（ツールチップ）を追加してアクセシビリティを向上させた。
