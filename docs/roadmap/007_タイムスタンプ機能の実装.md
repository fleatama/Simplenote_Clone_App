# 7. タイムスタンプ機能の実装

このアプリのオリジナル機能として、編集中のメモのカーソル位置に現在の日時を挿入する機能を実装する。

## 主な作業内容

1.  **UIの追加:**
    - 右ペインの編集エリアの上部に「タイムスタンプ挿入」ボタンを追加した。

2.  **カーソル位置への挿入ロジック:**
    - Reactの `useRef` フックを使い、テキストエリアのDOM要素への参照を作成。これにより、カーソル位置 (`selectionStart`) のような、Reactの状態管理だけでは取得しにくい情報を直接取得できるようにした。
    - ボタンがクリックされたら、`handleInsertTimestamp` 関数を実行。
    - この関数内で、現在のカーソル位置を取得し、その位置にフォーマットされた日時文字列を挿入するようにテキストを再構築。
    - 更新されたテキスト内容で `selectedNoteContent` 状態を更新し、画面に反映させた。
