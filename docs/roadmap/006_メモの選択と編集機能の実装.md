# 6. メモの選択と編集機能の実装

一覧からメモを選択し、その内容を編集できるようにする機能を実装した。

## 主な作業内容

1.  **選択状態の管理:**
    - `useState` を使い、選択されているメモのID (`selectedNoteId`) と内容 (`selectedNoteContent`) を保持する新しい状態を追加した。

2.  **選択機能の実装:**
    - メモ一覧の各アイテムに `onClick` イベントを設定し、クリックされたメモのIDと内容を状態に保存する `handleSelectNote` 関数を呼び出すようにした。
    - 選択されたアイテムには `active` クラスを付与し、見た目で選択状態が分かるようにした。

3.  **編集機能の実装:**
    - 右ペインのテキストエリアの `value` を `selectedNoteContent` 状態に紐づけ。
    - テキストエリアの `onChange` イベントで、入力された内容を `selectedNoteContent` 状態に反映させるようにした。これにより、テキストエリアが編集可能になった。（この時点では変更の保存はされない）

4.  **自動選択機能の実装:**
    - `useEffect` を使い、ページ読み込み時やメモリストが更新された際に、自動的にリストの最初のメモが選択されるようにした。
    * `handleCreateNewNote` 関数を修正し、新しいメモが作成された直後にそのメモが自動で選択されるようにした。
