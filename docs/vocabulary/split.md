# Split

## 汎用的な意味
- **意味:** 割る、裂く、分割する。
- **イメージ:** 1つの大きな塊を、真ん中から2つに分けること。

## IT業界での意味
- **意味:** **スプリット（分割表示）**。
- **使い方:** 
    - 画面を左右（または上下）に分けて、2つの内容を同時に表示すること。
    - Obsidianの「Live Preview」や、今回の「編集＋プレビュー」の並列表示がこれに当たります。
- **例文:** "The split-view allows you to see the preview while typing."（スプリット表示を使えば、入力しながらプレビューを確認できる。）
