# Unit Test

## 汎用的な意味
- **意味:** 単位テスト。
- **イメージ:** 大きな機械をいきなり動かすのではなく、まずネジ一本や部品一つが正しく動くかを確認すること。

## IT業界での意味
- **意味:** **単体テスト**。
- **使い方:** アプリケーション全体の大きな動きではなく、関数（Function）一つ一つなどの最小単位をテストすること。
- **メリット:** どこが壊れているのかを特定しやすく、実行速度も非常に速い。
- **例文:** "Let's start by writing unit tests for our utility functions."（まずは便利関数の単体テストから書き始めよう。）
