# GFM (GitHub Flavored Markdown)

## 汎用的な意味
- **意味:** GitHub風の味付けがされたマークダウン。
- **イメージ:** 標準的な料理（Markdown）に、GitHubが「使いやすさ」という調味料（機能）を加えた特製メニュー。

## IT業界での意味
- **意味:** **GitHub標準のマークダウン拡張仕様**。
- **特徴:** 
    - URLを貼るだけで自動的にリンクになる。
    - 表（テーブル）が書ける。
    - `- [ ]` でチェックリストが作れる。
    - `~~打ち消し線~~` が使える。
- **例文:** "We should enable GFM to support tables and checklists."（テーブルやチェックリストをサポートするためにGFMを有効にすべきだ。）
