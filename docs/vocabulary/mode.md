# Mode

## 汎用的な意味
- **意味:** 様式、形態、方法、気分。
- **イメージ:** その時の状況に合わせて、全体の雰囲気やルールが切り替わっている状態。

## IT業界での意味
- **意味:** **モード（動作モード）**。
- **使い方:** 
    - 同じアプリでも、機能や見た目を特定の目的に合わせて切り替えること。
    - 今回の場合、「編集に集中するモード」「プレビューで確認するモード」などを指します。
- **例文:** "You can switch between 'Source' and 'Reading' modes."（「ソース」モードと「閲覧」モードを切り替えることができる。）
