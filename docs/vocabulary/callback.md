# Callback

## 汎用的な意味
- **意味:** 折り返し電話。
- **イメージ:** 相手に一度用件を伝えて、相手が準備できたら自分に連絡をくれること。

## IT業界での意味
- **意味:** **（処理が終わった後に）呼び出される処理、またはURL**。
- **使い方:** 
    1. **Callback Function:** Aという関数の処理が終わった直後に自動で動くBという関数のこと。
    2. **Callback URL:** 今回の Google ログインのように、Google 側での「はい、ログインOKです！」という処理が終わった後に、Google がユーザーを連れ戻してくれる「アプリ側の住所（URL）」。
- **ニュアンス:** 「後でやっておいてね」という予約のようなイメージです。
