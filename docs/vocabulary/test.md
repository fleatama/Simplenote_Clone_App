# Test / Testing

## 汎用的な意味
- **意味:** 試験、検査、試すこと。
- **イメージ:** 本番の前に、それが本当に正しいか、壊れていないかを確認する作業。

## IT業界での意味
- **意味:** **テスト（検証作業）**。
- **使い方:** ソフトウェアが仕様通りに動くかを確認する全工程。
- **ニュアンス:** 「不具合（バグ）がないことを証明する」というよりは、「意図した通りに動いていることを確認する」という前向きな意味合いで使われます。
- **例文:** "We should run the automated tests before deployment."（デプロイ前に自動テストを実行すべきだ。）
