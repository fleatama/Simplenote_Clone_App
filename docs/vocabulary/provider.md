# Provider

## 汎用的な意味
- **意味:** 提供者、供給元。
- **イメージ:** 何か必要なサービスや物を、あなたに代わって準備してくれる人や会社。
- **例文:** "Internet Service Provider (ISP)"（インターネットを家まで繋いでくれる会社）。

## IT業界での意味
- **意味:** **（特定の機能を提供する）サービス元**。
- **使い方:** 今回の Google ログインの場合、Google が「認証情報をあなたのアプリに提供してくれる側」なので、Google のことを **「Auth Provider」** と呼びます。
- **ニュアンス:** 「どこが提供している機能か？」を指す時に使います（例：GitHub Provider, AWS Providerなど）。
