# Props

## 汎用的な意味
- **意味:** 演劇の小道具 (propertiesの略)。
- **イメージ:** 舞台の上で役者が使う「剣」や「カバン」のように、その役を演じるために渡される必要なアイテム。

## IT業界での意味
- **意味:** **プロップス（部品に渡すデータ）**。
- **使い方:** 
    - 親コンポーネントから子コンポーネントへ渡されるデータや設定。
- **今回のケース:** `NoteList` という部品に、「表示するメモの内容（`notes`）」や「今はローディング中か？（`loading`）」といった情報を外から渡しています。これらが Props です。
- **例文:** "We need to pass the selectedNoteId as a prop to the NoteList component."（選択されたノートIDをプロップスとしてNoteListコンポーネントに渡す必要がある。）
