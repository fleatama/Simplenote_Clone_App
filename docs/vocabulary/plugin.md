# Plugin

## 汎用的な意味
- **意味:** 差し込み用品、付け足しのもの。
- **イメージ:** 本体（掃除機など）に、用途に合わせて付け替える「ノズル」のようなもの。

## IT業界での意味
- **意味:** **プラグイン（拡張機能）**。
- **使い方:** 
    - すでにあるプログラムに、新しい機能を追加するための小さなプログラム。
- **今回のケース:** `remark-gfm` や `remark-breaks` がこれに当たります。`react-markdown` という本体に、これらを「差し込む」ことで、高機能な Markdown 処理が可能になります。
