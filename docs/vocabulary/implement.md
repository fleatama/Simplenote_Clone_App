# implement

## 汎用的な意味
- **意味:** （計画・決約・指示などを）実行する、実施する、履行する。
- **イメージ:** 頭の中にある計画や、紙に書かれた命令を、実際の行動に移して形にすること。
- **例文:** "The government will implement a new policy."（政府は新しい政策を実施する予定だ。）

## IT業界での意味
- **意味:** **実装（じっそう）する**。
- **使い方:** 設計書やアイデアを、実際にプログラムコードを書いて動くようにすること。
- **ニュアンス:** 単に「作る」よりも「仕様に基づいて、特定の機能を動作可能にする」という専門的な響きがあります。
- **例文:** "We need to implement the autosave feature by tomorrow."（明日までに自動保存機能を実装する必要がある。）
