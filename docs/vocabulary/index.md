# Index

## 汎用的な意味
- **意味:** 指標、索引、見出し。
- **イメージ:** 本の最後にある「索引」のように、特定の情報がどこにあるかを指し示す番号や記号。

## IT業界での意味
- **意味:** **インデックス、添字（そえじ）**。
- **使い方:** 
    - 配列（データのリスト）の中で、そのデータが何番目にあるかを表す数字。
    - プログラミングでは通常、1番目を **`0`** から数え始めます（0-based index）。
- **今回のケース:** プレビュー画面に複数のチェックボックスがある時、「上から何番目（どのインデックス）のチェックボックスか」を特定するために使います。
- **例文:** "We need to find the index of the clicked checkbox."（クリックされたチェックボックスのインデックスを特定する必要がある。）
