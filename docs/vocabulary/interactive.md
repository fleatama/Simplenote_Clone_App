# Interactive

## 汎用的な意味
- **意味:** 相互に作用する、対話的な。
- **イメージ:** 一方的に情報を受け取る（見るだけ）のではなく、こちらが何か操作すると、相手（システム）が反応を返してくれる状態。

## IT業界での意味
- **意味:** **インタラクティブ、対話型**。
- **使い方:** 
    - ユーザーの操作に応じてリアルタイムで画面が変化する機能。
    - 今回のケースでは、「プレビュー画面から直接データを操作できる（チェックを入れられる）」状態をインタラクティブな機能と呼びます。
- **例文:** "The user wants an interactive checklist in the preview mode."（ユーザーはプレビューモードで対話的なチェックリストを求めている。）
