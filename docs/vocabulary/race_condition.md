# Race Condition

## 汎用的な意味
- **意味:** 競争状態、競合状態。
- **イメージ:** 運動会の徒競走のように、複数の人が一つのゴール（資源）を目指して競い合っている状態。

## IT業界での意味
- **意味:** **競合状態（きょうごうじょうたい）**。
- **仕組み:** 2つ以上の処理が同時に同じデータにアクセスし、その「実行順序」によって結果が変わってしまう不具合のこと。
- **今回のケース:**
    1. 処理Aが「今の最大IDは5だ」と読み込む。
    2. 処理Bも（Aが終わる前に）「今の最大IDは5だ」と読み込む。
    3. Aが「ID 6」で保存する。
    4. Bも「ID 6」で保存し、Aのデータを上書きして壊してしまう。

## 解決策
- 今回のように「絶対に被らないID（UUID）」を使えば、そもそも順番を競う必要がなくなるため、問題が解決します。
