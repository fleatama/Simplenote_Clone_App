# Feature (Feat)

## 汎用的な意味
- **意味:** 特徴、機能、目玉商品。
- **イメージ:** その製品や作品が持っている「一番のウリ」や「特別な要素」。

## IT業界での意味
- **意味:** **機能、新機能**。
- **使い方:** 
    - ソフトウェアが提供する特定の動作や能力のこと。
    - また、Gitのコミットメッセージで、機能追加をした際のプレフィックスとして **`feat:`** と略して書くことが一般的。
- **例文:** "This app has a dark mode feature."（このアプリにはダークモード機能がある。）
- **関連:** `feat` は `feature` の略称。
