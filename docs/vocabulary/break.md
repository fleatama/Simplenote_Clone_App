# Break

## 汎用的な意味
- **意味:** 壊す、中断する、休憩。
- **イメージ:** 繋がっているものを途中で断ち切ること。

## IT業界での意味
- **意味:** **改行（Line Break）**。
- **使い方:** 
    - 文章を途中で切り、次の行へ移ること。
    - 今回導入する `remark-breaks` は、Markdownの「Enter1回は無視する」というルールを「壊して（Break）」、強制的に改行させる役割を持っています。
- **ニュアンス:** プログラミングでは `break` 文（ループを途中で抜ける処理）としても頻繁に使われる、超重要単語です。
