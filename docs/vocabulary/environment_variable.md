# Environment Variable

## 汎用的な意味
- **Environment:** 環境、周囲の状況。
- **Variable:** 変数、変わりやすいもの。

## IT業界での意味
- **意味:** **環境変数（かんきょうへんすう）**。
- **使い方:** データベースのパスワードやAPIのURLなど、プログラム本体に直接書きたくない（場所や環境によって変えたい）設定情報を、システム側で管理する変数のこと。
- **今回のケース:** `.env.local` ファイルに書いた `KV_REST_API_TOKEN` などがこれに当たります。
- **メリット:** コードを書き換えずに設定だけを変えられ、セキュリティも守れます。
