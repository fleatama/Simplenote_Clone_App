# Source

## 汎用的な意味
- **意味:** 源、源泉、情報源。
- **イメージ:** 川の始まりの場所や、噂の出処（でどころ）。

## IT業界での意味
- **意味:** **ソース（元のデータ、プログラムコード）**。
- **使い方:** 
    - 画面に表示されている結果（プレビュー）に対して、その元となっている文字データのこと。
- **今回のケース:** プレビュー画面のチェックボックスから見て、その元となったエディタ側の「テキストの行」を指します。
- **例文:** "We need to sync the preview with the source text."（プレビューとソーステキストを同期させる必要がある。）
