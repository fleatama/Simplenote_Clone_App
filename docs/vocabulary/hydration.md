# Hydrate / Hydration

## 汎用的な意味
- **意味:** 水分を補給する、潤す。
- **イメージ:** 乾燥した干し椎茸を水に浸して、元の形に戻すこと。

## IT業界での意味
- **意味:** **静的なHTMLにJavaScriptの命を吹き込むこと**。
- **仕組み:** Next.js（React）において、サーバーから届いた「ただの文字（HTML）」を、ブラウザ側で「動くボタンや状態を持つアプリ」として活性化させるプロセスのこと。
- **よく使われる用語:** **Hydration Mismatch**: サーバーが作った見た目と、ブラウザで作った見た目が食い違ってしまった時に出るエラー。
