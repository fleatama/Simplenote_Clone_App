# Component

## 汎用的な意味
- **意味:** 構成要素、部品、成分。
- **イメージ:** レゴブロックやプラモデルのパーツのように、全体を作り上げている一つ一つの「部品」。

## IT業界での意味
- **意味:** **コンポーネント（部品）**。
- **使い方:** 
    - 画面上の「ボタン」や「入力フォーム」、「ヘッダー」などを、再利用可能な独立したパーツとしてプログラムしたもの。
- **今回のケース:** `NoteList.tsx` は「メモ一覧を表示する」という役割を持った独立したコンポーネントです。
- **メリット:** 部品ごとにテストができるため、不具合を見つけやすくなり、修正も楽になります。
