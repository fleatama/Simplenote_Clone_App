# Auth (Authentication / Authorization)

## 汎用的な意味
- **意味:** 認証、権限、正当性の証明。
- **イメージ:** 「あなた、誰？」と「あなた、それをしていい人？」を確認すること。

## IT業界での意味
IT業界では、よく似た2つの言葉をまとめて「Auth」と呼ぶことが多いですが、実は意味が違います。

1. **Authentication（認証）:**
    - **意味:** 「本人確認」をすること。
    - **例:** ログイン（IDとパスワード）。
2. **Authorization（認可）:**
    - **意味:** 「アクセス許可」を与えること。
    - **例:** ログインはできたが、管理画面には入れない（一般ユーザー権限）。

## ニュアンス
開発者の会話では「Auth周りを直す（ログイン機能を直す）」のように、非常に頻繁に使われます。
