# saving / saved

## 汎用的な意味
- **save (動詞):** 救う、蓄える、節約する。
- **saving (現在分詞/名詞):** 救っている、蓄え、節約。
- **saved (過去形/過去分詞):** 救われた、蓄えられた。

## IT業界での意味
- **意味:** **保存中（saving） / 保存済み（saved）**。
- **使い方:**
    - **saving:** データをハードディスクやサーバーへ書き込んでいる真っ最中の状態。
    - **saved:** 書き込みが正常に完了し、データが確定した状態。
- **ニュアンス:** ユーザーに対して「今処理を頑張っていますよ」と伝えるのが `saving`、「もう安心ですよ」と伝えるのが `saved` です。
- **例文:**
    - "Please do not close the window while saving."（保存中にウィンドウを閉じないでください。）
    - "Your changes have been saved."（変更内容は保存されました。）
