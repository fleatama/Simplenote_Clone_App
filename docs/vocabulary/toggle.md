# Toggle

## 汎用的な意味
- **意味:** 切り替えスイッチ、留め木。
- **イメージ:** ダッフルコートのボタン（トグルボタン）のように、1つの操作で「オン」と「オフ」を交互に切り替えること。

## IT業界での意味
- **意味:** **トグル（状態の反転切り替え）**。
- **使い方:** 
    - チェックボックスをクリックして「チェックあり/なし」を切り替えること。
    - ダークモードのスイッチで「ライト/ダーク」を切り替えること。
- **ニュアンス:** 「AからBへ変える」だけでなく「今の状態から反対の状態へ変える」という、繰り返しの切り替え操作を指します。
