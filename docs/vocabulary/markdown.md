# Markdown

## 汎用的な意味
- **意味:** 記号を使って文書を装飾する軽量な書き方。
- **イメージ:** 複雑なワープロソフトを使わずに、普通のテキストに `#` や `*` を付け加えるだけで、見出しや太字を表現する手法。

## IT業界での意味
- **意味:** **マークダウン記法**。
- **使い方:** エンジニアがドキュメントを書く際の標準的な形式。GitHubやObsidian、そして今回のSimplenoteクローンでも中心となる技術です。
- **例文:** "Users want to write notes using Markdown syntax."（ユーザーはマークダウン記法を使ってメモを書きたいと考えている。）
