# Refactor / Refactoring

## 汎用的な意味
- **意味:** 再構築、再構成。

## IT業界での意味
- **意味:** **リファクタリング（コードの整理・改善）**。
- **使い方:** 「プログラムの動き（機能）」は変えずに、「コードの中身」をより綺麗に、読みやすく、テストしやすく書き換えること。
- **今回のケース:** `app/page.tsx` 内の関数を `utils/helpers.ts` に移動した作業がこれに当たります。
- **例文:** "I'm refactoring the NoteList component to improve testability."（テストしやすくするためにNoteListコンポーネントをリファクタリングしている。）
