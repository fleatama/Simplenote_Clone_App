# Debounce

## 汎用的な意味
- **意味:** （電気接点などの）バウンス（跳ね返り）を除去する。
- **イメージ:** 1回のつもりが何度も押されてしまうのを、「1回だけ」として正しく検知する仕組み。

## IT業界での意味
- **意味:** **デバウンス（実行頻度の制御）**。
- **使い方:** 短時間に何度も発生するイベント（文字入力など）を、最後の一回から一定時間経った後にだけ実行する手法。
- **今回のケース:** 「文字を打つたびに保存」するのではなく、「入力をやめて2秒後に保存」するようにしたのがデバウンスです。
- **メリット:** サーバーへの無駄な通信を劇的に減らすことができます。
