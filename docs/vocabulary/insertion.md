# insertion

## 汎用的な意味
- **意味:** 挿入、差し込み、書き込み。
- **イメージ:** 物と物の間に、別の何かを入れ込むこと。
- **例文:** "The insertion of a key into a lock."（鍵を鍵穴に差し込むこと。）

## IT業界での意味
- **意味:** **挿入（そうにゅう）**。
- **使い方:**
    1. テキストエディタなどで、特定のカーソル位置に文字を入れること。
    2. データベースに新しいデータを追加すること（SQLの `INSERT` 文など）。
- **ニュアンス:** すでにある流れやデータの間に、新しいものを割り込ませるイメージです。
- **例文:** "The cursor position determines the timestamp insertion point."（カーソルの位置によってタイムスタンプの挿入箇所が決まる。）
