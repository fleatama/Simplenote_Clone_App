# idle

## 汎用的な意味
- **意味:** 仕事をしていない、何もしていない、遊びの、暇な。
- **イメージ:** エンジンがかかっているが、車が走っていない状態（アイドリング）と同じです。
- **例文:** "The machines are idle during the holidays."（機械は休日の間、動いていない。）

## IT業界での意味
- **意味:** **待機状態（たいきじょうたい）**。
- **使い方:** システムやプログラムが動作可能だが、現在は特に何の処理も行っていない状態。
- **ニュアンス:** 「暇」というよりは「次の命令を待っている準備完了状態」というポジティブな意味合いで使われます。今回のアプリでは、保存処理が終わって落ち着いている状態を指します。
- **例文:** "The status returns to idle after the save is complete."（保存完了後、ステータスは待機状態に戻る。）
