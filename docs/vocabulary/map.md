# Map / Mapping

## 汎用的な意味
- **意味:** 地図、図図化する、対応づける。
- **イメージ:** 迷子にならないように、A地点とB地点を線で結んで対応関係を明らかにすること。

## IT業界での意味
- **意味:** **マッピング（対応付け）**。
- **使い方:** 
    - 異なる2つのデータの関連性を定義すること。
- **今回のケース:** 「画面上の3つ目のボタン」が「テキストの15行目」に対応していることを突き止める作業を **Source Mapping** と呼びます。
- **例文:** "Source mapping ensures the checkbox toggles the correct line."（ソースマッピングによって、チェックボックスが正しい行を切り替えることが保証される。）
