# Parser

## 汎用的な意味
- **意味:** 解析器。
- **イメージ:** 複雑な文章やデータを分解して、コンピュータが理解できる形に翻訳・整理するプログラム。

## IT業界での意味
- **意味:** **パーサー（解析プログラム）**。
- **使い方:** 今回の場合、ユーザーが書いた「ただの文字（Markdown）」を解析して、「HTML（Web用の部品）」に変換する役割を指します。
- **例文:** "We use react-markdown as a parser to display the preview."（プレビューを表示するために、パーサーとしてreact-markdownを使用する。）
