# Sanitize

## 汎用的な意味
- **意味:** 衛生的にする、消毒する、無害化する。
- **イメージ:** 汚れたものを洗って、安全に使える状態にすること。

## IT業界での意味
- **意味:** **サニタイズ（無害化処理）**。
- **重要性:** ユーザーが入力した文字の中に、悪意のあるプログラム（スクリプト）が混じっていることがあります。それをそのまま実行するとアプリが乗っ取られる危険があるため、危険なコードを無効化する処理が必要です。
- **例文:** "It's crucial to sanitize user input before rendering it as HTML."（HTMLとして表示する前に、ユーザー入力をサニタイズすることが極めて重要だ。）
