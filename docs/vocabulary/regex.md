# Regex (Regular Expression)

## 汎用的な意味
- **意味:** 正則表現。
- **イメージ:** 特定のパターンの文字列を、記号を使って表現する「魔法の検索式」。

## IT業界での意味
- **意味:** **正規表現（せいきひょうげん）**。
- **使い方:** 
    - 「メールアドレスっぽい形式」や「電話番号っぽい形式」などを一瞬で探し出すために使います。
    - 今回は、メモの中から `- [ ]` や `- [x]` という「チェックボックスのパターン」だけを探し出すために使用します。
- **ニュアンス:** 非常に強力ですが、書き方が独特（例: `/^(\d{3})-(\d{4})$/` など）なので、少しずつ慣れていくのがコツです。
- **例文:** "Use Regex to match the checkbox syntax in the text."（正規表現を使って、テキスト内のチェックボックス記法にマッチさせる。）
